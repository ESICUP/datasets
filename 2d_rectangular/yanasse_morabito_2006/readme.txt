DESCRIPTION:
This dataset contains 30 problem instances presented in:
Yanasse, H. H. and Morabito, R. (2006). Linear models for 1-group two-dimensional guillotine cutting problems. International Journal of Production Research,44:17,3471-3491. DOI: 10.1080/00207540500478603

FILE FORMAT:
- Each file represents a single instance.
- First line: object's length (L) and object's width (W)
- Second line: number of item types (m)
- In the following m lines: item type's length (li), item type's width (wi), number of demand/copies (bi).
